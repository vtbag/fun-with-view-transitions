Adding View Transitions to Your Starlight Site

The exercise site of this episode of Fun With View Transitions is a bit bigger. It is a real Starlight site.

See how to set it up on Starlight's Getting Started page.

You can then follow along the single steps from Episode 5, or you just add the Starlight Plugin that will do it all for you.

Caution:If you only want a cross-fade effect for the whole viewport, just enable basic view transitions and visit Episode 1 if you are looking for some background information.
